React To-Do List App
A simple and responsive To-Do List application built using React. This app helps users manage daily tasks efficiently by allowing them to add, delete, and track tasks.
 Features
* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 📱 Responsive design (works on mobile & desktop)
* ⚡ Fast and dynamic UI using React
  
Tech Stack
  Frontend: React.js
  styling:  CSS
  Version Control: Git & GitHub

 Installation & Setup
1. Clone the repository
git clone  https://github.com/anushareddyv1234-bit/Todo-list.git

2. Navigate to the project folder
cd Todo-List

3. Install dependencies
npm install

4. Start the development server
npm start

 How It Works
* Users can enter a task in the input field
* Click "Add" to store it in the list
* Tasks can be marked as completed
* Tasks can be removed when no longer needed
